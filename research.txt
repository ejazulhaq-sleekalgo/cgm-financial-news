## Implementation Code for WordPress
When writing PHP inside WordPress, never use curl or file_get_contents directly. WordPress provides a built-in wrapper framework called wp_remote_get(). It natively handles timeouts, errors, and securely caches results using the [WordPress Transients API](https://developer.wordpress.org/apis/transients/) so you don't burn through your 4 daily free news API requests on every page refresh. [6, 7, 8, 9, 10] 
You can paste this custom function into your theme's functions.php file or use a code snippets plugin: [11, 12] 

function get_eodhd_custom_news() {
    // 1. Use WordPress Transients to cache data for 12 hours (43200 seconds)
    // This protects your Free Plan daily API limit from being exhausted instantly
    $cached_news = get_transient('eodhd_ticker_news');
    if ( $cached_news !== false ) {
        return $cached_news;
    }

    // 2. Define API Parameters
    $api_token = 'YOUR_FREE_API_TOKEN'; // Replace with your real token
    $symbols   = 'AKEMF.US,NIOMF.US,ZCTSF.US,NZAUF.US,DSEAF.US,ZUUZF.US';
    
    // 3. Construct URL
    $url = add_query_arg( array(
        's'         => $symbols,
        'limit'     => 10,
        'api_token' => $api_token,
    ), 'https://eodhd.com' );

    // 4. Execute safe WordPress remote request
    $response = wp_remote_get( $url, array( 'timeout' => 15 ) );

    // Return empty array if request fails
    if ( is_wp_error( $response ) ) {
        return array();
    }

    // 5. Decode Response
    $body = wp_remote_retrieve_body( $response );
    $data = json_decode( $body, true );

    // 6. Cache the successful result before returning
    if ( is_array( $data ) ) {
        set_transient( 'eodhd_ticker_news', $data, 43200 ); 
        return $data;
    }

    return array();
}

## Displaying the News in Your Theme Template
To render the pulled articles cleanly on your website frontend, loop over the array payload like this:

$articles = get_eodhd_custom_news();

if ( ! empty( $articles ) ) {
    echo '<ul class="eodhd-news-list">';
    foreach ( $articles as $article ) {
        $title = esc_html( $article['title'] );
        $link  = esc_url( $article['link'] );
        $date  = esc_html( date( 'M j, Y', strtotime( $article['date'] ) ) );
        $stock = esc_html( implode( ', ', $article['symbols'] ) );

        echo "<li>";
        echo "<a href='{$link}' target='_blank'><strong>{$title}</strong></a><br>";
        echo "<small>Published: {$date} | Tickers: {$stock}</small>";
        echo "</li>";
    }
    echo '</ul>';
} else {
    echo '<p>No recent news found for these assets.</p>';
}
